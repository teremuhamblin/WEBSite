À propos

Cette page présente la vision, le contexte et la philosophie du projet WEBSite.

Vision

WEBSite est conçu comme un squelette minimaliste, extensible, documenté, pour lancer rapidement des projets web structurés sans sacrifier la qualité technique.

Principes

  • Structure claire et prévisible
  • Design System centralisé
  • Documentation dès le début
  • Scripts simples mais robustes
