# 📁 src/
>Dossier principal contenant le code source du projet.

### Structure
- /*.html — Pages principales du site  
- assets/ — Images, icônes, polices et ressources statiques  
- css/ — Feuilles de style du projet  
- js/ — Scripts JavaScript

### Objectif
Centraliser toutes les pages et ressources nécessaires au rendu du site.

### Bonnes pratiques
- Garder les pages HTML simples et cohérentes  
- Organiser les styles et scripts par modules si nécessaire  
- Placer uniquement les ressources utilisées dans assets/  
- Maintenir une structure claire pour faciliter l’évolution du projet
