# css/

Contient les feuilles de style du projet.

## Contenu
- `main.css` — Styles globaux
- `layout.css` — Mise en page
- `components.css` — Styles des composants
- `variables.css` — Couleurs, typographies, variables

## Objectif
Assurer une organisation claire et modulaire des styles.

## Bonnes pratiques
- Favoriser la modularité
- Utiliser des classes explicites
- Centraliser les variables pour une cohérence visuelle
