# assets/

Contient toutes les ressources statiques utilisées par le site.

## Contenu
- Images (PNG, JPG, SVG)
- Icônes
- Polices
- Médias divers

## Objectif
Centraliser les ressources pour garder une structure claire et éviter la duplication.

## Bonnes pratiques
- Utiliser des noms de fichiers descriptifs
- Organiser en sous-dossiers si nécessaire
- Optimiser les images avant intégration
