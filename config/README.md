# config/

Ce dossier contient les fichiers de configuration du projet **WEBSite**.  
Ils centralisent les paramètres du projet, les chemins, les modules activables et les métadonnées.

## 📁 Contenu

- `website.conf` — configuration générale (serveur, version, déploiement)
- `paths.conf` — chemins principaux du projet
- `modules.conf` — activation/désactivation des modules
- `project.conf` — métadonnées et options globales

Les scripts du projet chargent automatiquement ces fichiers.
